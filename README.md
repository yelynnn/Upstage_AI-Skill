# stock-briefing-cli

> 투자 성향(공격형/안정형)에 맞는 일일 주식 브리핑 CLI

yfinance로 6개월치 OHLCV 데이터와 연간 재무제표를 가져와 **기술적 지표 + 재무 건전성**을 계산하고,
Google News RSS로 실시간 뉴스를 수집한 뒤, **Upstage Solar Pro 3** AI가 세 관점을 종합해
매매 시그널과 근거를 생성합니다.

---

## 설치

```bash
npm install -g stock-briefing-cli
```

> **시스템 요구사항**
> - Node.js 14+
> - Python 3 — 기본 경로: `/opt/anaconda3/bin/python3`
>   - 경로가 다르면: `export STOCK_BRIEFING_PYTHON=/usr/bin/python3`
> - Python 패키지 (최초 1회):
>   ```bash
>   pip install yfinance requests
>   ```

## 빠른 시작

```bash
# 1. AI 시그널 API 키 설정 (선택, 없어도 지표는 동작)
export UPSTAGE_API_KEY=your_key_here   # https://console.upstage.ai

# 2. 투자 프로필 설정 (최초 1회)
stock-briefing setup

# 3. 일일 브리핑 실행
stock-briefing run

# 4. 급변 알림 체크
stock-briefing alert
```

## 커맨드 상세

### `stock-briefing setup` — 투자 프로필 설정

인터랙티브 대화형으로 프로필을 구성합니다. 기존 프로필이 있으면 덮어쓸지 선택할 수 있습니다.

**① 투자 성향 선택**
```
투자 성향을 선택해주세요:
  1. 공격형  — 높은 수익 추구, 리스크 적극 감수
  2. 안정형  — 안정적 수익 추구, 리스크 최소화

선택 (1/2):
```

**② 관심 종목 입력** — 아래 방식 중 자유롭게 혼합 사용

| 입력 | 처리 방식 | 예시 |
|------|-----------|------|
| 종목명 | DB 자동 매핑 | `삼성전자`, `SK하이닉스`, `NAVER` |
| 분야명 | 대표 종목 리스트 추천 | `반도체`, `배터리`, `바이오`, `AI` |
| 6자리 코드 | KOSPI 자동 처리 | `005930` → `005930.KS` |
| KOSDAQ 코드 | 그대로 처리 | `035720.KQ` |
| 해외 티커 | 대문자 변환 | `AAPL`, `NVDA`, `TSLA` |

**지원 분야** (분야명 입력 시 대표 종목 목록 제공):
반도체 / 배터리 / 2차전지 / 전기차 / 바이오 / 헬스케어 / 제약 / IT / 게임 / 플랫폼 /
금융 / 은행 / 보험 / 증권 / 자동차 / 에너지 / 화학 / 통신 / 철강 / 소재 / 유통 / 건설 / 항공 / 해운 / AI / 방산


## 기술적 지표

| 지표 | 계산 방법 | 시그널 기준 |
|------|-----------|-------------|
| **이동평균선** | 5 / 20 / 60 / 120일 MA | 정배열(단기>장기) / 역배열 / 혼조 |
| **골든·데드크로스** | MA5 vs MA20 교차 (최근 3일 이내) | 골든크로스=매수 신호 / 데드크로스=매도 신호 |
| **눌림목** | MA20 ±2% 이내 + 이전 10일 상승 추세 | 매수 진입 타이밍 |
| **볼린저밴드** | 20일 2σ | 상단근접(과매수) / 하단근접(과매도) / 중간(0~100%) |
| **이격도** | 현재가 / MA20 × 100 | ≤98 → 매수권 / ≥105 → 매도권 |
| **캔들패턴** | 최근 3일 분석 | 상승장악형·망치형·적삼병·샛별형(매수) / 하락장악형·흑삼병·저녁별형(매도) / 도지 |
| **지지·저항** | 최근 20일 저점/고점 | 지지선 이탈 시 매도 경고 |
| **거래량** | 20일 평균 대비 배수 | ≥2배 → 급증🔥 / ≥1.3배 → 증가 / ≤0.5배 → 감소 |
| **추세** | 20일 선형회귀 기울기 | 기울기 +0.3% 이상→상승 / −0.3% 이하→하락 / 그외→횡보 |
| **박스권** | 20일 고저 범위 10% 이내 | 횡보 구간 경고 |

## 재무 건전성 지표

| 항목 | 지표 | 판정 기준 |
|------|------|-----------|
| **안정성** | 유동비율 | ≥100% 정상 / <100% 주의 / <50% 위기경고 |
| **안정성** | 당좌비율 | <30% → 유동성 위기 |
| **안정성** | 부채비율 | ≤200% 정상 / >300% 위험 |
| **수익성** | ROE | ≥15% 우량 / ≥5% 보통 / <0% 자본잠식 가능 |
| **수익성** | 영업이익률 + 매출성장률 | 동반 양수 → 성장 중 |
| **현금흐름** | 영업(+) / 투자(−) / 재무(−) | 우량패턴 여부 |

**이상신호 자동 감지:**

| 이상신호 | 패널티 |
|----------|--------|
| 영업현금흐름(−) + 순이익(+) → 가짜이익 가능성 | −25점 |
| 매출채권 전년대비 +30% 이상 급증 → 부실매출 가능성 | −15점 |
| 재고자산 전년대비 +30% 이상 급증 | −15점 |
| 유동비율 <50% → 유동성 위기 | −30점 |
| 당좌비율 <30% → 유동성 위기 | −20점 |
| 부채비율 >200% | −15점 |
| ROE <0% → 자본잠식 가능성 | −20점 |
| 매출 증가 + 영업이익(−) → 원가 압박 | −15점 |

**재무 종합 등급:** A(우량, 85점↑) / B(보통, 70점↑) / C(주의, 50점↑) / D(위험)

> 안정형 투자자는 이상신호 감지 시 B등급도 C로 하향 조정되며 AI 시그널에 반영됩니다.

## AI 매매 시그널 기준

### 공격형

| 시그널 | 조건 |
|--------|------|
| **강매수** | 골든크로스 + 정배열 + 볼린저 하단 + 거래량 급증 |
| **매수** | 눌림목 + 이격도 ≤98 + 양봉패턴 |
| **주목** | MA 수렴 + 박스권 상단 돌파 시도 |
| **관망** | 위 조건 해당 없음 |
| **매도** | 데드크로스 + 역배열 + 볼린저 상단 |
| **강매도** | 흑삼병 + 역배열 + 거래량 급증 하락 + 재무 이상신호 |

### 안정형

| 시그널 | 조건 |
|--------|------|
| **매수** | 정배열 + 이격도 ≤98 + 지지선 근처 + 유동비율 ≥100% + 부채비율 ≤200% |
| **주목** | 골든크로스 후 눌림목 확인 |
| **관망** | 박스권 횡보·추세 불명확 / 재무 이상신호 감지 시 무조건 관망 |
| **매도** | 데드크로스 또는 지지선 이탈 |
